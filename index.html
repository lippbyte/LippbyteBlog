Setup GitHub Connection

Admin panel ini menggunakan GitHub API untuk menyimpan konten langsung ke repository kamu. Ikuti langkah berikut:

1 Buka github.com/settings/tokens/new dan buat Personal Access Token (Classic)
2 Centang scope repo (full control of private repositories)
3 Copy token-nya dan paste di bawah. Token hanya disimpan di browser kamu (localStorage), tidak dikirim ke server manapun.
⚠️ Jangan share token ini ke siapapun. Kalau repo lo public, pastikan token hanya punya akses ke repo yang diperlukan saja.
